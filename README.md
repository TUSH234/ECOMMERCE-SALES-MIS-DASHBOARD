# E-Commerce Sales & Operational Performance MIS Dashboard

## 📊 Project Overview
This repository contains an interactive, industry-grade **Sales & Operational Performance MIS Dashboard** built using advanced spreadsheet analytics. The project processes and models a complex retail dataset containing over **1,200+ rows of raw transactional records** to extract actionable business insights and monitor key volume and value indicators dynamically.

The primary objective of this project is to convert unstructured transactional data into high-level executive insights, allowing stakeholders to monitor revenue trends, track operational logistics, and evaluate marketing acquisition channels dynamically across multiple fiscal years (including Year-to-Date analysis up to June 2025).

## 🛠️ Core Features & Analytics Architecture

### 1. Advanced Data Modeling & ETL
* **Data Cleansing:** Audited raw transactional logs, handled missing values, and standardized date/text formats across 1,200+ rows.
* **Feature Engineering:** Formulated automated gross revenue matrices by mapping granular volume metrics with dynamic unit prices.
* **Temporal Dimensioning:** Segmented raw timestamps into structured fiscal dimensions (Years, Quarters, and Months) to facilitate precise Year-over-Year (YoY) and Year-to-Date (YTD) comparisons.

### 2. Interactive UI/UX Design & Control Engine
* **Application Skin:** Designed a clean, minimalist premium dark-themed environment. Gridlines were disabled to replicate a dedicated business intelligence software interface.
* **Cross-Functional Slicers:** Implemented centralized interactive temporal slicers (Year and Quarter filters).
* **Multi-Table Report Connections:** Engineered a logical data backend where a single slicer control seamlessly filters and updates all independent charts and KPI blocks simultaneously.

### 3. Executive Visualization Layers
* **Year-over-Year Performance Matrix:** Utilizes a clustered line layout to evaluate seasonal trends and growth vectors across 2023, 2024, and 2025 (YTD).
* **Marketing Acquisition Analysis:** A horizontal bar chart isolating transaction volumes and revenue contributions coming through Search Engines, Instagram, Facebook, and Email marketing.
* **Logistical Supply Chain Tracker:** A clean donut chart layout displaying real-time operational status (Delivered, Pending, Returned, and Cancelled rates) to optimize fulfillment channels.
* **Dynamic KPI Summaries:** Automated top-level metric cards tracking **Total Revenue**, **Total Order Volumes**, and **Cumulative Quantities** that instantly respond to any applied multi-level filter.

## 📈 Key Business Insights Extracted
* **Marketing Optimization:** Identified the most lucrative customer acquisition channels, providing stakeholders with clear data to optimize marketing budget allocations.
* **Supply Chain Efficiency:** Categorized operational failure points by visualizing the exact proportions of returned and cancelled orders, allowing for proactive inventory and logistical adjustments.
* **Trend Analysis:** Isolated seasonal surges and transactional drop-offs using multi-year matrix tracking, improving demand-forecasting capabilities.
